# CodeAlpha_Music-Player
# 🎵 Pro Music Player (JavaScript Project)
A modern and feature-rich **Music Player Web Application** built using **HTML, CSS, and JavaScript**.
This project simulates a mini Spotify-like experience with advanced controls and interactive UI.

---

## 🚀 Features

* ▶️ Play / Pause songs
* ⏮ Previous / ⏭ Next controls
* 🎧 Animated Equalizer
* 🔁 Shuffle mode
* 🔂 Repeat mode
* ❤️ Add to Favorites (saved using LocalStorage)
* 🔍 Search songs in playlist
* 🎚 Progress bar (seek control)
* 🔊 Volume control
* 📀 Active song highlighting
* 🎨 Modern Spotify-style UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Animations)
* JavaScript (DOM Manipulation, Events, LocalStorage)

## ⚙️ How to Run

1. Download or clone this repository
2. Open the folder in VS Code
3. Right-click `index.html`
4. Click **Open with Live Server**

---

## ⚠️ Note on Songs

* This project uses **free demo audio files** for testing
* Due to copyright restrictions, real songs are not included
* You can add your own songs by updating the `songs` array in `script.js`

Example:

```
src: "songs/your-song.mp3"
```
